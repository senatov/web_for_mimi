A native macOS 26+ file manager for real file work

MiMiNavigator is built for Mac users searching for a Finder replacement, a dual-pane file browser, a Total Commander style workflow, a signed DMG file manager, an SFTP/SMB client, an archive manager, or a practical tool for cleaning and organizing large folders. It brings together a Finder-style sidebar, Google Drive share links, automatic cloud drive detection, GPS geotag detection on photos, smart copy with conflict resolution, persistent selection restore, archive browsing, and a disk ballast finder.

Dual-panel file browsing

Two side-by-side panels with independent navigation, tabs, favorites, and history. Drag files between panels, compare directories at a glance, or just keep your source and destination visible at the same time. Useful when Finder windows, copy dialogs, and breadcrumb paths slow you down.

SFTP, FTP & SMB — all built in

Connect to remote storage without leaving the two-panel workspace. SFTP, FTP, and SMB locations open inside the panels, so local and remote paths can be browsed, copied, and compared side by side. Saved connections use status indicators, Keychain-backed credentials, startup auto-connect, and history-based restore after disconnects. SMB shares can be managed even when system /Volumes access is not available.

Archive manager for ZIP, RAR, 7Z, TAR and more

Open ZIP, RAR, 7Z, TAR, GZIP, BZIP2, XZ, ISO, CAB, XAR, LZH, ARJ, and CPIO archives and browse them like regular directories. Nested archives are supported too. Create archives with progress tracking, cancellation, and clear context-menu actions instead of jumping to a separate utility.

Advanced file search

Find files by name, date range, age, size, and type. Search by modification or access date, filter by deletable-only, and use the built-in “ballast finder” to locate old unused files eating your disk. Real-time results with smart pruning that skips macOS bundle containers and runtime junk.

Keyboard-first workflow

Hotkeys for everything — F2 for inline rename, F5 for copy, F6 for move, F7 for new folder. Tab switches panels. Insert marks files. Multi-selection with ⌘-Click, ⇧-Click, and range select. Total Commander users will feel right at home.

GeoTag detection on photos

Photos with GPS coordinates get a small globe badge on their icon — automatically, in the background, after every directory scan. Reads EXIF headers from JPEG, HEIC, TIFF, DNG, and 10+ RAW formats without decoding pixels. See at a glance which shots have location data.

Finder-style sidebar

A toggleable source list with favorites, recent history, mounted volumes, AirDrop, Network, Trash, and a dedicated Cloud Drives group. MiMiNavigator scans every folder in ~/Library/CloudStorage and auto-detects Google Drive, OneDrive, Dropbox, Proton Drive, iCloud Drive, Box, and pCloud. New cloud providers are picked up automatically.

Selection memory that follows your path

Go into a folder, select a file, step back to the parent, and return later: MiMiNavigator remembers the recent selection and scrolls it back into view. It keeps the last navigation hops per panel, so deep photo folders and cloud directories no longer reset you to the top of the list.

Smart copy & conflict handling

File operations show live byte-level progress. When a conflict appears, a side-by-side dialog compares the existing and incoming file by size, date, and preview. Skip or replace per file, or apply the same decision to the whole operation. macOS service metadata such as .DS_Store and Spotlight leftovers are excluded automatically, and Mac app packages are copied as package directories instead of fragile flat file streams.

Export any file to the cloud in one click

One-click cloud export with public link

Select any file in either panel, right-click, and export it straight to Google Drive. The upload runs in the background with a live progress indicator; when it finishes, MiMiNavigator creates a public share link and copies it to your clipboard. No browser tabs, no drag-and-drop to a web page, no manual permission hunt — one context-menu action turns a local file into a shareable cloud link for documents, archives, images, videos, and other Google Drive-supported files.

Private long links for real sharing

Google Drive share links without manual browser work

Right-click a selected file or folder and choose Copy Share Link. MiMiNavigator uploads it to Google Drive Public, creates an external link, and copies it to the clipboard with View only or Allow editing access. The first use opens Google OAuth; after that, the refresh token lives in macOS Keychain and repeated share-link actions run without reopening the browser. Uploaded items land in My Drive/Public, and the synced folder appears as Google Drive Public in the sidebar.

Built-in Media Format Converter

Built-in media converter

Right-click a video, audio, or image file, choose Convert Media, pick a target format, and run the conversion without leaving the file manager. Everyday paths include MP4 to AVI, WAV to MP3, PNG to JPEG, MKV to MOV, FLAC to AAC, WEBM to MP4, TIFF to PNG, and OGG to WAV.

Supported video, audio, and image formats

Video: MP4, AVI, MKV, MOV, WEBM, FLV, WMV. Audio: MP3, WAV, FLAC, AAC, OGG, WMA, M4A, AIFF. Images: PNG, JPEG, TIFF, BMP, WEBP, GIF, HEIC, SVG.

No separate app and no workflow break: convert files from the same place where you browse, preview, move, and organize them.

Why build another file manager for macOS?

Because after 20 years of macOS, Apple still thinks one panel is enough. I grew up on Total Commander, moved to Mac, and no file browser came close. So I wrote my own dual-panel file manager from scratch.

What makes this macOS file manager different?

Moving files without losing your place. This file browser gives you two panels, tabs, a Finder-style sidebar, SFTP/FTP/SMB, archive formats as folders, smart copy with conflict cards, Google Drive share links, remembered selections, GPS geotag detection on photos, a disk ballast finder, and a keyboard workflow that does not fight you.

Can I use it instead of Finder on macOS 26+?

Yes. MiMiNavigator is designed as a native Finder alternative for macOS 26 or newer: dual-pane browsing, tabs, Finder-style sidebar, SFTP/FTP/SMB, archive folders, search, preview, media conversion, cloud share links, and file operations are available from one signed desktop app.

Roadmap: features worth copying from successful competitors

The next product direction is focused on workflows that already made other macOS file managers valuable: batch rename with live before/after preview and undo, a persistent preview pane, folder sync and directory compare, a command bar for quick actions, saved workspaces, Git status badges, stronger direct remote/cloud backends such as WebDAV/S3/B2, duplicate search, app cleanup, and deeper Finder/system integration. The goal is not to add noise; it is to finish the file-manager jobs that power users pay for elsewhere.

How do I get it?

Download the signed DMG above, open it, and drag MiMiNavigator to Applications. Launch it from your Applications folder and start working right away. No App Store, no subscription, no extra installer. The source code is on GitHub too.

Is it really free?

Yes. No trial, no subscription, no hidden fees. MiMiNavigator is open source under AGPL-3.0 and distributed as a signed and notarized DMG. Use it forever.

What macOS version do I need?

{{ minimumMacOSVersion }} or newer. Built natively with SwiftUI and AppKit — no Electron, no web wrappers, no Java runtime. Pure macOS performance.

Can I share files via cloud directly?

Yes. Right-click any file, choose Export to Google Drive, and a public share link is copied to your clipboard automatically. No browser needed after the first OAuth login.

Can I contribute?

Absolutely. The full source code is on GitHub. File issues, submit pull requests, or just star the repo if you find it useful. Feedback always welcome.
