Features

Built for the jobs Finder makes expensive

MiMiNavigator combines commander-style speed with modern macOS UI: real two-panel work, remote locations, archive browsing, media inspection, cloud sharing, and search workflows in one signed app.

01

Panel workflow

Two independent panels, tabs, favorites, history, Finder-style sidebar, remembered selection, drag batches, and shared List/Preview/Tree view state.

  • Tab switches panels; Insert marks and moves to next file.
  • Preview mode now uses the same marked-file state as List mode.
  • Tree mode adds lazy expandable directory rows with metadata columns.
02

Remote and cloud

SFTP, FTP, and SMB locations open inside the panels, while Share+Link works with mounted Google Drive and Dropbox accounts.

  • Cloud drives are detected from ~/Library/CloudStorage.
  • Google Drive supports view-only and editable links; Dropbox supports view-only links.
  • Dropbox uses OAuth PKCE, stores refresh tokens in Keychain, and creates its Public folder when needed.
  • Provider links become short branded MiMiNavigator URLs and copy directly to the clipboard.
03

Search and archives

Advanced search covers names, content, size, age, dates, deletable-only ballast scans, and archive contents without leaving the file manager.

  • Browse ZIP, RAR, 7Z, TAR, DMG, ISO, CAB, XAR, LZH, ARJ, CPIO, and more.
  • Archives behave like virtual folders with parent navigation.
  • Large directories use timeouts, cache fallback, and sampled autofit.
04

Media and metadata

Media Info, EXIF photo dates, GPS map links, geotag badges, and Convert Media workflows are part of the file workflow.

  • Convert video, audio, and image files from the context menu.
  • Photos with GPS metadata get a compact badge on their file icon.
  • Preview and metadata stay available while navigating siblings.

FAQ and roadmap

Practical answers for a practical file manager

MiMiNavigator is free, open source, signed, notarized, and still moving quickly.

Why build another file manager for macOS?

Finder is good for casual browsing, but serious file work needs two panels, keyboard shortcuts, visible paths, batch operations, and fewer windows.

Can it replace Finder on macOS 26+?

Yes for file-manager work: dual-pane browsing, tabs, sidebar, SFTP/FTP/SMB, archives, search, preview, media conversion, and Google Drive or Dropbox Share+Link are all in one app.

How do I install it?

Download the signed DMG, open it, drag MiMiNavigator to Applications, and launch it. No App Store and no extra installer.

Is it really free?

Yes. No trial, subscription, or hidden fee. The source is available under AGPL-3.0.

Roadmap

Next direction: batch rename with live before/after preview, folder sync and directory compare, saved workspaces, command bar, Git status badges, duplicate search, app cleanup, WebDAV/S3/B2, and deeper Finder/system integration.
