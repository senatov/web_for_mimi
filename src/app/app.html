Built-in Media Format Converter

One-click conversion between any media formats — video, audio, images

MiMiNavigator includes a built-in media converter that transforms any media file into any other format with a single click. Convert MP4 to AVI, WAV to MP3, PNG to JPEG, MKV to MOV, FLAC to AAC, WEBM to MP4, TIFF to PNG, OGG to WAV — and dozens more combinations. No extra software needed, no command-line tools to install. Right-click any file, pick the target format, done.

Supported conversions include all common video formats (MP4, AVI, MKV, MOV, WEBM, FLV, WMV), audio formats (MP3, WAV, FLAC, AAC, OGG, WMA, M4A, AIFF), and image formats (PNG, JPEG, TIFF, BMP, WEBP, GIF, HEIC, SVG). The best free media file converter built right into your macOS file manager.

Dual-panel file browsing

Two side-by-side panels with independent navigation, tabs, and history. Drag files between panels, compare directories at a glance, or just keep your source and destination visible at the same time.

SFTP & FTP that just works

Connect to any remote server with one click. Saved connections with colored status indicators, Keychain-stored passwords, auto-connect on startup. Browse remote files exactly like local ones.

15+ archive formats as folders

Open ZIP, RAR, 7Z, TAR, GZIP, BZIP2, XZ, ISO, CAB, XAR, LZH, ARJ, CPIO — browse them like regular directories. Nested archives too. Create archives with progress tracking and cancel support.

Built-in media converter

Right-click any video, audio, or image file and convert it to any other format with one click. MP4, AVI, MKV, MOV, MP3, WAV, FLAC, PNG, JPEG, HEIC, and dozens more — no extra software needed.

Advanced file search

Find files by name, date range, size, and type. Recursive search across directories with real-time results. Smart filtering that skips macOS bundle containers to show you only what matters.

Keyboard-first workflow

Hotkeys for everything — F2 for rename, F5 for copy, F6 for move, F7 for new folder. Tab switches panels. Total Commander users will feel right at home.

Why build another file manager for macOS?

Because after 20 years of macOS, Apple still thinks one panel is enough. I grew up on Total Commander, moved to Mac, and no file browser came close. So I wrote my own dual-panel file manager from scratch.

What makes this macOS file manager different?

Moving files without losing your mind. This file browser gives you two panels, tabs, SFTP that just works, 15+ archive formats as folders, and a keyboard workflow that does not fight you.

How do I get it?

Download the signed DMG above, open it, and drag MiMiNavigator to Applications. Launch it from your Applications folder and start working right away. No App Store, no subscription, no extra installer. The source code is on GitHub too.

Is it really free?

Yes. No trial, no subscription, no hidden fees. MiMiNavigator is open source under AGPL-3.0 and distributed as a signed and notarized DMG. Use it forever.

What macOS version do I need?

macOS {{ minimumMacOSVersion }} or newer. Built natively with SwiftUI and AppKit — no Electron, no web wrappers, no Java runtime. Pure macOS performance.

Can I contribute?

Absolutely. The full source code is on GitHub. File issues, submit pull requests, or just star the repo if you find it useful. Feedback always welcome.
