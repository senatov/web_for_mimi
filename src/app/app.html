The file manager macOS should have shipped. I built it.

Built-in Media Format Converter

One-click conversion between any media formats — video, audio, images

MiMiNavigator includes a built-in media converter that transforms any media file into any other format with a single click. Convert MP4 to AVI, WAV to MP3, PNG to JPEG, MKV to MOV, FLAC to AAC, WEBM to MP4, TIFF to PNG, OGG to WAV — and dozens more combinations. No extra software needed, no command-line tools to install. Right-click any file, pick the target format, done.

Supported conversions include all common video formats (MP4, AVI, MKV, MOV, WEBM, FLV, WMV), audio formats (MP3, WAV, FLAC, AAC, OGG, WMA, M4A, AIFF), and image formats (PNG, JPEG, TIFF, BMP, WEBP, GIF, HEIC, SVG). The best free media file converter built right into your macOS file manager.

Why build another file manager for macOS?

Because after 20 years of macOS, Apple still thinks one panel is enough. I grew up on Total Commander, moved to Mac, and no file browser came close. So I wrote my own dual-panel file manager from scratch.

What makes this macOS file manager different?

Moving files without losing your mind. This file browser gives you two panels, tabs, SFTP that just works, 15+ archive formats as folders, and a keyboard workflow that doesn’t fight you.

How do I get it?

Download the signed DMG above, open it, and drag MiMiNavigator to Applications. Launch it from your Applications folder and start working right away. No App Store, no subscription, no extra installer. The source code is on GitHub too.
