Dual-panel file manager for macOS

MiMiNavigator, or simply MiMiNav, is a dual-panel file manager for practical work on macOS. It combines tabs, favorites, history and FTP/SFTP access in a clearer and faster workflow than Finder.

FAQ

What is MiMiNavigator for macOS?

I never really found a file manager I liked enough to keep, except perhaps Total Commander, so I built my own. And yes, I am still improving it.

FAQ

What can it do?

It helps you work with files faster using two panels, tabs, favorites, history and FTP/SFTP access in a cleaner workflow than Finder.
